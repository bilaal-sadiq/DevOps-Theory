Introduction to Linux Families

  • Dark blue: text typed at the command line
  • Green: Output
  • Black: File content
  • Brown: File/Directory names

Red Hat Family Systems

  • Fedora is an upstream testing platform
  • CentOS is a close clone of RHEL
  • Intelx86, ARM, Itanium, PowerPC
  • yum package manager
  • Widely used by enterprises

SUSE Family Systems

  • SLES is upstream for openSUSE
  • RPM-based zypper package manager
  • Uses YaST for system administration
  • Widely used in retail

Debian Family Systems

  • Debian is upstream for Ubuntu
  • DPKG-based APT package manager
  • Widely used for cloud deployments
  • GNOME-based, visually distinct

Linux Philosophy and Concepts

  • Kernel: Glue between hardware and apps (e.g. Linux kernel)
  • Distribution: Complete OS (e.g. Ubuntu, Fedora)
  • Boot loader: Starts OS (e.g. GRUB)
  • Service: Background process (e.g. httpd, ntpd)
  • File System: Storage structure (e.g. ext4, XFS)
  • X Window System: Graphical base layer
  • Desktop Environment: User interface (e.g. GNOME, KDE)
  • Command Line: Text interface
  • Shell: Command interpreter (e.g. bash, zsh)
